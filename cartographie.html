Mode d'emploi

• Glissez une carte pour la déplacer.
• Sous le nom, ajoutez une note (version, qui l'utilise, remarque…).
• Tirez la pastille bleue (bord droit) vers une autre carte pour créer une flèche.
• Cliquez une flèche pour la sélectionner, son libellé pour le nommer.
• Suppr efface l'élément sélectionné.
• Tout est enregistré dans ce navigateur.
